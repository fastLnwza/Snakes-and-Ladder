# สรุปลำดับการแสดงข้อความของ Minigames

## 1. Precision Timing Game (QTE) - ช่อง 7, 55

**ลำดับการแสดงข้อความ:**
1. เริ่มเกม: `"Press SPACE to stop at 4.99!"`
2. ระหว่างรอ: `"4.99:X.XX [space!]"` (แสดงเวลาเพิ่มขึ้น)
3. เมื่อกด SPACE:
   - แสดง: `"4.99 : X.XX"` (เปรียบเทียบ target กับเวลาที่กด) - แสดง 0.5 วินาที
   - แล้วแสดงผล:
     - ถ้าถูกต้อง (±0.01): `"โบนัส +6"`
     - ถ้าผิด: `"Mission Fail"`

## 2. Tile Memory Game - ช่อง 28, 82

**ลำดับการแสดงข้อความ:**

**Round 1 (3 ตัวเลข):**
1. แสดงลำดับ: `"mem X _ _"` → `"mem X Y _"` → `"mem X Y Z"` (แสดงทีละตัว)
2. รอให้ผู้เล่นพิมพ์: `"input _ _ _"`
3. เมื่อพิมพ์ถูกต้อง: ไป Round 2
4. เมื่อพิมพ์ผิดหรือหมดเวลา: `"MEM FAIL"` (แสดง 2 วินาที)

**Round 2 (4 ตัวเลข):**
1. แสดงลำดับ: `"mem A _ _ _"` → ... → `"mem A B C D"` (แสดงทีละตัว)
2. รอให้ผู้เล่นพิมพ์: `"input _ _ _ _"`
3. เมื่อพิมพ์ถูกต้อง: `"MEM SUCCESS +4"` (แสดง 2 วินาที)
4. เมื่อพิมพ์ผิดหรือหมดเวลา: `"MEM FAIL"` (แสดง 2 วินาที)

## 3. Reaction Minigame (Number Guessing) - ช่อง 12, 48

**ลำดับการแสดงข้อความ:**

1. เริ่มเกม: `"Guess 1-9"` (แสดง 3 วินาที)
2. รอให้ผู้เล่นเดา: `"Guess 1/3 : input"`
3. เมื่อเดาเลข:
   - แสดง: `"Guess 1/3 : input 5"` (แสดง 2 วินาที)
   - ถ้าถูกต้อง (ครั้งที่ 1-2): `"Too low"` หรือ `"Too high"` (แสดง 1 วินาที)
   - ถ้าถูกต้อง (ครั้งที่ 3): ข้าม feedback ไปแสดง `"Mission failed"` ทันที
   - ถ้าถูกต้อง: `"You win! +3 steps"` (แสดง 2 วินาที)
4. หลังจาก feedback (ถ้าไม่ใช่ครั้งสุดท้าย): `"Guess 2/3 : input"` แล้ววนซ้ำ
5. ถ้าเดาผิดครบ 3 ครั้ง: `"Mission failed"` (แสดง 2 วินาที)

---

## 4. Math Quiz - ช่อง 33, 66

**ลำดับการแสดงข้อความ:**

1. เริ่มเกม: `"Math: X + Y = ? (1-9) Time: 15s"`
2. ระหว่างรอ: `"Math: X + Y = ? (1-9) Time: Xs"` (อัพเดทเวลาที่เหลือทุกเฟรม)
3. เมื่อตอบ:
   - ถ้าถูกต้อง: `"Correct! +4 steps"`
   - ถ้าผิด: `"Wrong Answer!"`
   - ถ้าหมดเวลา: `"Time's Up!"`

---

## 5. Pattern Matching Game - ช่อง 18, 72

**ลำดับการแสดงข้อความ:**

1. เริ่มเกม: `"Pattern: ↑ ↓ ← →"` (แสดง 3 วินาที)
2. รอให้ผู้เล่นทำตาม: `"Repeat pattern! (WASD)"`
3. เมื่อตอบ:
   - ถ้าถูกต้องครบ: `"Perfect! +5 steps"`
   - ถ้าผิด: `"Wrong Pattern!"`
   - ถ้าหมดเวลา: `"Time's Up!"`

---
